◆ _new_string()

string* _new_string ( )

Erstellt einen neuen leeren String und gibt das Ergebnis zurück. Im Fehlerfall (z. B. kein Speicher verfügbar), wird das Programm beendet. Der neue string* muss am Ende gefreit werden.

Dies ist eine "interne" Funktion, die nur innerhalb der strlib aufgerufen werden sollte.

Returns
string* Der neue leere String.
