◆ _new_string()

string * _new_string ( )

Erstellt einen neuen leeren String und gibt das Ergebnis zurück. Im Fehlerfall (z. B. kein Speicher verfügbar), wird das Programm beendet.

Dies ist eine "interne" Funktion, die nur innerhalb der strlib aufgerufen werden sollte.

Rückgabe
string* Der neue leere String.
